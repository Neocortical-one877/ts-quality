# Purpose

This repo exists to provide an offline-first quality platform for TypeScript/JavaScript changes where trust is earned by explicit evidence rather than intuition.

## In scope
- deterministic evidence gathering and artifact generation
- invariant evaluation over focused, aligned test evidence
- explainable merge-confidence, governance, and legitimacy surfaces
- additive-first artifact/report contracts that downstream tools and reviewers can inspect

## Out of scope
- semantic guesswork presented as proof
- repo-global keyword coincidence treated as behavioral evidence
- workflow/control-plane features that do not strengthen this repo's native evidence model
